Recent Projects

GA University Student Portal

A hackathon project, this student portal has built-in accessibility features

Project: GA University Student Portal

GA Hackathon App was an app developed during a General Assembly-sponsored hackathon. The hackathon's theme was internet accessibility. The end product was a student portal through which a student could access various items such as course syllabi and grades as well as contact his/her instructor. Keep in mind the emphasis was accessibility thus some features of the portal would not have been completely built out.

Technologies employed:
React.js, auth0 login authentication, HTML, CSS, Bootstrap, Netlify
Github repository:
https://github.com/Gingaling/GAhackathon-spring2022
Kitchen Counter

An inventory of your groceries for a more efficient shopping experience...

Project: Kitchen Counter

Kitchen Counter is a food inventory application. At a minimum, it tracks what foods a user currently has in their cupboard or refrigerator. It also permits a user to record what they eat. With these two pieces of information the user will know when they are running low on a food item. Their food shopping will thus be more efficient and will eliminate those trips to the supermarket because one has forgotten that one product they meant to buy.

Technologies employed:
Javascript, React.js, Mongoose, Node.js, Express, MongoDB, Heroku, Netlify, HTML5, CSS and Animate.css
Github repositories:
https://github.com/Gingaling/eat-n-shop-frontend (frontend);
https://github.com/Gingaling/eat-n-shop-api (backend)
Book Roulette

A Trio of Cats

Are you a cat person? Then this random cat image generator is just your thing.


Project: A Trio of Cats

A Trio of Cats is a single-page application created with ReactJS and styled utilizing Flexbox. The app appropriately uses the useState hook, a new feature in the most recent React upgrade. The app incorporates the use of a fetch operation to a third party API and uses retrieved data. In this case, it was random cat images. Further the app demonstrates the use of calling a component with state via an onClick event.

Technologies employed:
Javascript, React.js (including React Router and ReactDOM), JSX, vanilla Javascript, Flexbox and Netlify
Github repository:
https://github.com/Gingaling/a-trio-of-cats-3
